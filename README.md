# Learning Progress Dashboard

A simple local HTML dashboard for tracking learning progress across multiple topics and projects.

This project is built as a single `index.html` file and runs directly in the browser.  
No server, no installation, and no account are required.

## What it does

- Track multiple learning paths and sub-topics
- Set progress from 1 to 10
- Add current status and next steps
- Search and filter topics
- Switch between English and Hebrew
- Save data automatically in the browser using localStorage
- Export and import JSON backups

## Who is it for?

This can be useful for students, self-learners, developers, creators, or anyone learning several things at the same time and wanting a simple visual way to stay organized.

## How to use

1. Download or clone this repository.
2. Open `index.html` in your browser.
3. Edit the topics and progress.
4. Use **Export JSON** from time to time to create your own backup.

## Important note

The data is saved locally in your browser.  
If you clear browser cache/site data, your saved progress may be deleted.  
Use **Export JSON** regularly if you want to keep backups.
